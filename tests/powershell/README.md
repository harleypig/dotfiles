# PowerShell tests

Pester tests for the repo's PowerShell side (e.g. `ps-startup.ps1`,
`powershell/`).

- Files: `*.Tests.ps1`
- Run: `pwsh -c 'Invoke-Pester tests/powershell'`

See `.claude/TESTS.md` for this repo's test layout and
`config/claude/rules/testing.md` for the multi-language test conventions.
